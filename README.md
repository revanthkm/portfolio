# Revanth Krishna - Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Live Demo

Visit the live site: [https://revanth802.github.io/portfolio](https://revanth802.github.io/portfolio)

## ✨ Features

- **Modern Design**: Clean, dark-themed UI with gradient accents
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Scroll animations, hover effects, and transitions
- **Contact Form**: Integrated with Formspree for message handling
- **Fast Loading**: Optimized assets and efficient code

## 🛠️ Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Google Fonts (Inter)

## 🚀 Deployment

This site is deployed on **GitHub Pages** (free hosting).

### To Deploy:

1. Create a new repository on GitHub named `portfolio`
2. Push this code to the repository
3. Go to Settings → Pages
4. Select "Deploy from a branch"
5. Choose "main" branch and "/ (root)" folder
6. Click Save

Your site will be live at: `https://[your-username].github.io/portfolio`

## 📧 Contact Form Setup

The contact form uses Formspree. To make it work:

1. Sign up at [formspree.io](https://formspree.io)
2. Create a new form
3. Replace `YOUR_FORM_ID` in the form action URL in index.html
4. Test the form submission

## 📝 Customization

### To Update Content:

1. **Personal Info**: Edit text in `index.html`
2. **Colors**: Modify CSS variables in `:root` in `styles.css`
3. **Projects**: Update project cards in the Projects section
4. **Skills**: Modify skill cards in the Skills section

### To Add a Profile Photo:

1. Add your photo to the project folder
2. Replace the placeholder div in the About section with an `<img>` tag
